# 🎬 IMDb Clone – React Movie App

A fully responsive movie web application built using React that allows users to explore trending movies and manage their watchlist.

## 🚀 Features

- 🔍 Browse trending movies (TMDB API)
- ❤️ Add/Remove Watchlist
- 🔎 Search functionality
- 🎭 Genre filtering
- 📊 Sort by rating & popularity
- 📄 Pagination
- 💾 LocalStorage persistence

## 🛠️ Tech Stack

- React.js
- Tailwind CSS
- Axios
- React Router
- TMDB API
